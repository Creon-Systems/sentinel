# Sentinel

A discrete event simulation (DES), this simulator focuses on multi-agent systems in different (logistics) domains for stress-testing control policies under correlated disruptions and adversarial conditions. This simulator can be used for any general graph simulation to analyse where catastrophic failures occur and how safe control policies are.
